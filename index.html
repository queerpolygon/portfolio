Aristotle’s To-Do App

This To-Do List application uses HTML, JavaScript, CSS, and React. It allows users to effortlessly manage their tasks, offering features such as task creation, editing, marking as complete, and removal. Additionally, I integrated a Random Quote Generator API to provide users with motivational or thought-provoking quotes, enhancing the overall user experience. The responsive design and polished interface make it an accessible and visually appealing app. See on Github

What’s Your DJ Name?

This fun and interactive web page uses Javascript to generate unique DJ names. Users can click a button to generate random DJ names instantly, and the stylish CSS design ensures an engaging and visually appealing user interface. See on Github

Power Bi Report - The Everything Company

In response to the urgent request from The Everything Company's Head of Finance, Hardik, I have developed a BI dashboard aimed at providing crucial insights into the company's product line performance. This dashboard, created using data analysis techniques, offers a comprehensive view of key performance indicators (KPIs) tailored to meet Hardik's specific objectives. It provides insights and recommendations to identify areas for potential improvement and cost-saving opportunities.

Power Bi Report - Executive Summary

I've created a concise report using the provided finance workbook, summarizing the company’s latest sales figures. The report highlights the most profitable month and year, the most successful countries/regions, and provides recommendations for product and segment investments. This data-driven report assists our decision-making process for continued success.
